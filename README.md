# Boilerplate

> HTML5 Boilerplate adaptado para trabalhar com Bower e Grunt.

### Linguagens

- Node: NPM, Grunt e Bower
- Ruby: Sass

### Como utilizar

Baixe as dependências do NPM

```shell
npm install
```

Baixe as dependências do Bower

```shell
bower install
```

Ative o Grunt

```shell
grunt w
```
